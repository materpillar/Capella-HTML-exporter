Initiate Audio-Video Stream

PhysicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Physical Architecture > Physical Functions > Root Physical Function > IFE System > Manage Audio Video Diffusion > Broadcast Stored Audio and Video Streams > Initiate Audio-Video Stream

This function is in charge of initializating a streaming operation. It reads the file(s) to be streamed in the digital media library and prepares the streaming header that is going to be sent to the target player (either seat TV or cabin screen). The stream header basically includes all information required to perform the decoding of the streams.
