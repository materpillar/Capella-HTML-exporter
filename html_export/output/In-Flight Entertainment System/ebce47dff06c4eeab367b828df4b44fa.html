Provide Navigation Data

LogicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Logical Architecture > Logical Functions > Root Logical Function > Provide Aircraft Information, Commands and Means > Provide Navigation Data

Navigation is the determination of position and direction on or above the surface of the Earth. Avionics can use satellite-based systems (such as GPS and WAAS), ground-based systems (such as VOR or LORAN), or any combination thereof. The IFE system requires this data to run the moving-map service.
