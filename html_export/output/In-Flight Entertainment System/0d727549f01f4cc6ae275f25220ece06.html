Retrieve Imposed Video Data

LogicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Logical Architecture > Logical Functions > Root Logical Function > IFE System > Run Services > Run Cabin Intercommunication Service > Retrieve Imposed Video Data

This function consists in receiving media information requests (list of imposed videos, available languages, etc.) and returning the data corresponding to these requests. This data is retrieved from the digital media storage unit.
