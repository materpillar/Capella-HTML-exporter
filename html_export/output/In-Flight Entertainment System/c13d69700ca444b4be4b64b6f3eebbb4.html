Determine Operating Profiles

PhysicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Physical Architecture > Physical Functions > Root Physical Function > IFE System > Determine Operating Profiles

Based on the notifications it receives from the function interfacing with the aircraft, this function computes the mode in which the IFE system has to operate. For example, if the aircraft sends a power reduction request, the IFE may switch to a downgraded mode.
