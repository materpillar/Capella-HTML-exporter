Capture VOD Selections

LogicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Logical Architecture > Logical Functions > Root Logical Function > IFE System > Run Services > Run Video-On-Demand Service > Capture VOD Selections

This function handles the interactions with the passenger in the VOD service. The Seat TV displays interaction widgets (typically buttons and slide bars). This typically airline-specifc, because user interface typically is. This function is responsible for capturing these selections and forwarding them to a more generic function (a controler) that will transform the selection into an internal system command.
