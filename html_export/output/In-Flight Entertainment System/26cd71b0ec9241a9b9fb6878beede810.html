Run Video-On-Demand Service

SystemFunction

In-Flight Entertainment System > In-Flight Entertainment System > System Analysis > System Functions > Root System Function > IFE System > Run Services > Run Video-On-Demand Service

VOD Service is a passenger service. This function gathers all functions in charge of managing the VOD-related interactions of the passenger (selection of movies, play/pause/stop controls, display of movie data, etc.).
