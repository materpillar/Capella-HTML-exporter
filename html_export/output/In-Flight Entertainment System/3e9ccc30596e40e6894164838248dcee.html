Store Passenger Service State

LogicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Logical Architecture > Logical Functions > Root Logical Function > IFE System > Manage Passenger Services Lifecycle > Store Passenger Service State

Whenever a service is interrupted, its current state has to be saved, in order to be able to resume it later on in the same state. This function typically delegates this action to the functions directly responsible for running each passenger service.
