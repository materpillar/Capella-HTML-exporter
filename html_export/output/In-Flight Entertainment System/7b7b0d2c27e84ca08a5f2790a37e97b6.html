Run Cabin Intercommunication Service

SystemFunction

In-Flight Entertainment System > In-Flight Entertainment System > System Analysis > System Functions > Root System Function > IFE System > Run Services > Run Cabin Intercommunication Service

Cabin intercommunication is a cabin crew service. This function gathers all functions in charge of managing the cabin crew interactions related to imposed videos (selection of videos, play/pause/stop controls, display of video data, etc.).
