Manage Passenger Service Interruptions

SystemFunction

In-Flight Entertainment System > In-Flight Entertainment System > System Analysis > System Functions > Root System Function > IFE System > Manage Video and Audio Diffusion > Manage Passenger Service Interruptions

This function is responsible for triggering passenger service interruptions as soon as an imposed video is played or an audio announcement is performed. When a VOD movie is running, this means for example that the movie as to be automatically paused and the current position stored.

Involving Functional Chains

  • Watch Imposed Video on Private Screen
  • Broadcast Audio Announcement
  • Start Playing VOD Movie
  • Resume VOD Movie
