Run Cabin Intercommunication Service

PhysicalFunction

In-Flight Entertainment System > In-Flight Entertainment System > Physical Architecture > Physical Functions > Root Physical Function > IFE System > Run Services > Run Cabin Intercommunication Service

Cabin intercommunication is a cabin crew service. This function gathers all functions in charge of managing the interactions of the cabin crew related to imposed videos (selection of videos, play/pause/stop controls, display of video data, etc.).
